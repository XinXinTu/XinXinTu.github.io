hexo+github+hexo-theme-matery搭建个人博客

hexo github node hexo-theme-matery 博客搭建
Publish Date:   2020-05-21

Windows搭建自己的博客

B站主页

步骤

软件配置

  1. 安装node.js；地址：https://nodejs.org/en/ ；选择LTS即可

  2. 安装后利用

    node -v

    查看node版本；

    npm -v

    查看包管理器的版本

  3. 安装git；地址：https://git-scm.com/downloads

hexo框架安装

  1. 利用镜像源下载安装cnpm

    npm install -g cnpm --registry=https://registry.npm.taobao.org

  2. 查看cnpm版本

    cnpm -v

  3. 安装hexo

    cnpm install -g hexo-cli

  4. 验证是否成功

    hexo -v

    搭建个人博客

  5. 建立并进入文件夹blog

    mkdir myblog
cd myblog/

  6. 已经安装好git后，打开blog所在文件夹，右键git bash即可

  7. 生成博客-启动博客

    hexo init
hexo s

  8. 进入端口地址，可以看到你的博客，切忌用ctrl+c

  9. 新建一个博客

    hexo n "我的第一个博客"

  10. 进入blog文件夹下source下_posts里面找到你新建的博客，用typora打开写入你想要展示的内容。

  11. 编辑完后回到git bash输入

    hexo clean
hexo g
hexo s

  12. 上面代码分别输入，分别代表清理博客-生成博客-预览博客

部署到github上

  1. github注册账号 新建仓库 注意仓库名与昵称保持一致，并记住这个账户名和密码 后面要用

  2. 安装git插件

    cnpm install hexo-deployer-git --save

  3. 修改博客根目录下_config.yml文件，windows直接用vs code打开即可，最好不用记事本

  4. # Deployment
# Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: git@github.com:XinXinTu/XinXinTu.github.io.git
  branch: main

  1. 进行远端部署

    hexo clean
hexo g
hexo s
hexo d

  2. 刷新仓库，可以看到blog下的文件被更新到了仓库下了，也可以通过此步骤来检测前面的步骤是否正确。’

  3. 最后进入github仓库，点击上方服务选择Github Pages，下方强制https最好勾上。

  4. 配置好后点击github pages弹出网址即可预览到你的博客。

主题设置

  1. 找到对应主题所在的地址

    git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

  2. 将其克隆后，修改_config.yml文件里的theme为对应名字，例如：yilia（也就是一成功后多余的文件夹名称）

    hexo clean
hexo g
hexo s
hexo d

    注意点

1.在最后 hexo s 打开博客时，本地端口不能用ctrl+c复制网址，否则会停止该操作，导致端口打不开，直接在网页输入即可。

2.同时github上步骤很详细，按照步骤完全没有问题，注意区分站点下的config文件与theme下的config文件修改是不一样的，链接如下：https://github.com/blinkfox/hexo-theme-matery
