Vue常用特性

1.过滤器

（1）过滤器的作用是什么？

处理数据，比如将字符串格式化为首字母大写，将日期化为指定的格式等。

数据流经Vue过滤器，经过处理之后满足我们特定需求。

（2）自定义过滤器？
Vue.filter('过滤器名称',function(value){
            // 过滤业务逻辑
        })
（3）过滤器的使用？
<div>{{msg | upper}}</div>
        <div>{{msg | upper | lower}}</div>
        <div v-bind:id = "id |formatId"></div>
（4）案例（数据首字母大写）

代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div id = "app">
        <input type="text" v-model = "msg">
        <div>{{msg | upper}}</div>
    </div>
    <script type = "text/javascript" src="vue.js"></script>
    <script type = "text/javascript">
        // 过滤器
        Vue.filter('upper',function(val) {
            return val.charAt(0).toUpperCase() + val.slice(1);

        })
        var vm = new Vue({
            el:"#app",
            data:{
                msg:''
            }
        });
        
    </script>
</body>
</html>

JavaScript charAt() 方法

（1）定义和用法：

charAt() 方法可返回指定位置的字符。

第一个字符位置为 0, 第二个字符位置为 1,以此类推.

（2）语法：

string.charAt(index)

（3）实例：

var str = "HELLO WORLD";
var n = str.charAt(2)

n的输出结果为：

L
JavaScript toUpperCase() 方法

（1）定义和用法

toUpperCase() 方法用于把字符串转换为大写。

（2）语法

stringObject.toUpperCase()

（3）返回值

一个新的字符串，在其中 stringObject 的所有小写字符全部被转换为了大写字符。

（4）实例

在本例中，”Hello world!” 将以大写字母来显示：

<script type="text/javascript">

var str="Hello World!"
document.write(str.toUpperCase())

</script>

运行结果：HELLO WORLD!

JavaScript slice() 方法

（1）定义和用法

slice() 方法可提取字符串的某个部分，并以新的字符串返回被提取的部分。

（2）语法

stringObject.slice(start,end)

（3）返回值

一个新的字符串。包括字符串 stringObject 从 start 开始（包括 start）到 end 结束（不包括 end）为止的所有字符。

（4）案例

在本例中，我们将提取从位置 6 开始的所有字符：

<script type="text/javascript">

var str="Hello happy world!"
document.write(str.slice(6))

</script>

输出：

happy world!
（5）局部过滤器：

全局过滤器和局部过滤器的区别是全局过滤器在全局的任何一个地方都可以使用，局部过滤器只在本组件中可以使用。

Vue实例本身也是一个组件，局部过滤器只有在组件的内部可以被使用。

filters:{
                upper:function(val){
                return val.charAt(0).toUpperCase() + val.slice(1);

                }

案例：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div id = "app">
        <input type="text" v-model = "msg">
        <p>级联：首字母大写再转化为首字母小写：</p>
        <div>{{msg | upper | lower}}</div>
        <p>首字母大写：</p>
        <div>{{msg | upper}}</div>
        <div :abc ="msg | upper">测试数据</div>


    </div>
    <script type = "text/javascript" src="vue.js"></script>
    <script type = "text/javascript">
        // 过滤器
        // Vue.filter('upper',function(val) {
        //     return val.charAt(0).toUpperCase() + val.slice(1);

        // })
        Vue.filter('lower',function(val) {
            return val.charAt(0).toLowerCase() + val.slice(1);

        })
        var vm = new Vue({
            el:"#app",
            data:{
                msg:''
            },
            filters:{
                upper:function(val){
                return val.charAt(0).toUpperCase() + val.slice(1);

                }
            }
        });
        
    </script>
</body>
</html>

（6）过滤器总结

Vue过滤器主要用于处理数据，格式化数据。从使用层面来说使用方式可以是全局的也可以用局部方式来定义。使用方式有两个小细节一个是可以使用到插值表达式中，也可是使用到属性绑定中。并且还支持级联操作。

（7）带参数的过滤器
Vue.filter('format',function(value,arg1) {
            // value就是过滤器传递过来的参数

        })
（8）带参数的过滤器的使用
<div>{{date | format('yyyy-MM-dd')}}</div>

使用过滤器的时候后面可以传参，括号里传递过来的参数是从第二个参数位置进行传参的，因为第一个参数默认是要处理的date数据

（9）案例（格式化日期）
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <div id="app">
    <div>{{date | format('yyyy-mm-dd hh:mm:ss')}}</div>
    </div>
    <script type = "text/javascript" src="vue.js"></script>

    <script>
        Vue.filter('format', function(value, arg) {
            function dateFormat(date, format) {
                if (typeof date === "string") {
                    var mts = date.match(/(\/Date\((\d+)\)\/)/);
                    if (mts && mts.length >= 3) {
                        date = parseInt(mts[2]);
                        }
                     }
                date = new Date(date);
                if (!date || date.toUTCString() == "Invalid Date") {
                    return "";
                    }
            var map = {
                "M": date.getMonth() + 1, //月份 
                "d": date.getDate(), //日 
                "h": date.getHours(), //小时 
                "m": date.getMinutes(), //分 
                "s": date.getSeconds(), //秒 
                "q": Math.floor((date.getMonth() + 3) / 3), //季度 
                "S": date.getMilliseconds() //毫秒 
            };

            format = format.replace(/([yMdhmsqS])+/g, function(all, t) {
                var v = map[t];
                if (v !== undefined) {
                    if (all.length > 1) {
                        v = '0' + v;
                        v = v.substr(v.length - 2);
                        }
                    return v;
                    } 
                else if (t === 'y') {
                            return (date.getFullYear() + '').substr(4 - all.length);
                        }
                return all;
                });
                return format;
                }
                return dateFormat(value, arg);
            })
        var vm = new Vue({
            el:'#app',
            data:{
                date:new Date()
            }
        });

    </script>
</body>
</html>

输出：

2021-45-23 17:45:20
（10）带参数过滤器总结

带参数过滤器的用法，使用过滤器的时候通过括号的方式传递参数。接收参数的时候从第二个参数进行接收，第一个参数是data值。

日期格式化规则

y：年 ，

M：年中的月份（1-12），

d：月份中的天（1-31） ，

h：小时（0-23），

m：分（0-59），

s：秒（0-59），

S：毫秒（0-999），

q：季度（1-4）

将时间格式化为yyyy-MM-dd hh：mm：ss
